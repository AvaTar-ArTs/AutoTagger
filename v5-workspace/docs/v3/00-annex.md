# V3 — The Annex (narrative layer)

**V3 playbooks** (`../playbooks/v3/`) are a deliberately different **creative skin**: same operational commitments as V2, expressed as an “archival annex” — labels, dossiers, and specimen cards.

## Authority

- **Procedures and commands:** `../docs/v2/` remains canonical.
- **V3 pages:** mood, sequencing, metaphor, and scenario framing.

If anything conflicts, **trust `docs/v2/`** and file a fix for the HTML or `LINK_MAP`.

## When to use V3

- Public demos where you want a memorable visual story.
- Portfolio pieces where “documentation as exhibit” matters.
- Workshops: open V3 first, drop to V2 when live-typing commands.
