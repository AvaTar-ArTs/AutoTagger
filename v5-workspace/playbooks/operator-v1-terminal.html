§ Quick start

First-time setup

cd /path/to/AutoTagger/v5-workspace
./setup_autotag.sh

Scan a folder you choose

./autotag.sh "/path/to/target" my_analysis --no-open

Default source = workspace root

./autotag.sh
# uses v5-workspace as source; timestamped run name

Compat pipeline: ./autotag_compat.sh with the same arguments.

§ Code examples

Batch multiple targets

#!/bin/bash
set -e
cd /path/to/AutoTagger/v5-workspace

./autotag.sh "/path/to/Automations" automations_scan --no-open
./autotag.sh "/path/to/Docs" docs_scan --no-open
./autotag.sh "/path/to/Clients/ClientA" client_a_scan --no-open

Rank tools from CSV (Python)

import pandas as pd

csv_path = "output/automations_scan/automations_scan_results.csv"
df = pd.read_csv(csv_path)
top = df.sort_values("predicted_business_value", ascending=False).head(20)
print(top[["name", "intelligent_category", "predicted_business_value"]])

Weekly artifact

RUN_NAME="weekly_$(date +%Y%m%d)"
./autotag.sh "/path/to/BusinessAssets" "$RUN_NAME" --no-open

§ Sellable bundles (reference)

  • Starter kit — scripts + sample outputs + README. Anchor $19–$49.
  • Pro bundle — configs, taxonomy presets, advanced docs. $79–$199.
  • DWY setup — live session + first-run handoff. $250–$1,000.

§ Distribution channels

  • Gumroad — tiers, lead magnet, email follow-up.
  • Codester — zip + changelog + verify script output screenshots.
  • Services — Upwork/Fiverr listings that use AutoTag as fulfillment.

§ Weekly metrics

  • Traffic → checkout conversion → AOV.
  • Support tickets per 100 sales.
  • Refund rate by tier.
  • Time-to-first successful scan (onboarding quality).

Low conversion → add proof: real CSV excerpts, screenshots, 2-minute screen recording.
