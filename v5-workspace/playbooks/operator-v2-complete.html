1) Quick Start

Setup

cd /path/to/AutoTagger/v5-workspace
./setup_autotag.sh

Run chosen source

./autotag.sh "/path/to/target" my_analysis --no-open

Run base source

./autotag.sh
# defaults to v5-workspace as source

2) Code Examples

Batch run multiple folders

#!/bin/bash
set -e
cd /path/to/AutoTagger/v5-workspace
./autotag.sh "/Users/you/Automations" automations_scan --no-open
./autotag.sh "/Users/you/Docs" docs_scan --no-open
./autotag.sh "/Users/you/Clients/ClientA" client_a_scan --no-open
