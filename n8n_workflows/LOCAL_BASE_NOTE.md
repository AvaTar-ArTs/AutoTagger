# AutoTagger Local n8n Base

This folder is the stable AutoTagger-local base path:

- **V2 docs** (`docs/v2/`, `CHANGELOG.md`, updated `README.md`) live here and are versioned with the AutoTagger repo when not gitignored.
- Workflow JSON trees (`free`, `premium`, `workflows`, `imports`) may symlink to:
  - `/Users/steven/n8n_workflows` (see `CANONICAL_HOME.md`)

Use this path in AutoTagger scripts to avoid breakage:

- `/Users/steven/AutoTagger/n8n_workflows`
